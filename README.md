# Face Recognition Attendance System 🎓

A real-time Face Recognition Attendance System built with Python.

## Built With
- Python 3.12
- OpenCV
- DeepFace
- Pandas

## Features
- Real-time face detection
- Student face registration
- Automatic attendance marking
- Saves attendance to CSV with time

## How to Run
1. Register student: `py -3.12 register_face.py`
2. Mark attendance: `py -3.12 attendance.py`
